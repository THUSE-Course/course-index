# Django 后端小作业文档

## 作业目标

本作业目标为使用基于 Python 语言的 Django 框架实现前述需求文档的 API 后端。本作业中所完成的 API 后端，应当可以与前端小作业中所完成的 Next.js 前端对接，完整构成一个带有记录管理功能的康威生命游戏。

相信大家在《程序设计训练》课程中对 Django 框架有了或多或少的了解，本次小作业旨在帮大家快速回忆起相关内容，并对《软件工程》课程实践中的一些重点内容进行概括性的讲解。

## 代码地址
